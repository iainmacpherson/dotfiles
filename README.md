# dotfiles
Repository of custom configs for various tools.

## Things required
* Neovim > 0.7
* pynvim (through pip3)
* zsh & oh-my-zsh
* Starship
* Tmux
* Nodejs >= 14.14.0

### Neovim plugin dependencies
* Telescope Fuzzy Finder
    * make
    * gcc
    * ripgrep (for string search in workspace)
* Mason    
    * npm
* A nerd font
